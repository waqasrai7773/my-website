## 🌐 How to Go Live (Free)

### Option 1 — GitHub Pages (Recommended)
1. Create a free account at [github.com](https://github.com)
2. Create a new repository named `waqasrai.github.io`
3. Upload all files in this folder
4. Go to Settings → Pages → Enable GitHub Pages
5. Your site is live at `https://waqasrai.github.io` 🎉

### Option 2 — Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop this entire folder
3. Done — instant live link!

## ✏️ Customization

- **Social links**: Edit the `href="#"` values in `index.html` with your GitHub/LinkedIn URLs
- **Projects**: Update project titles & descriptions in the `#projects` section
- **Colors**: Edit CSS variables at the top of `style.css` (`:root { ... }`)
